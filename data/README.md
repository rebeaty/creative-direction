# Data

Processed files needed to reproduce the manuscript and SI analyses.

## File Map

### `sample1/`

- `study1_process_20260224_221431.csv` — process-side ratings for the chat condition, with transcripts and the three model-specific creative-direction ratings.
- `creativity_scored_20260225_063157.csv` — product-side creativity scores from the three scoring models.

Join key: `user_id`, `task`, `condition`

### `sample2/`

- `study2_process_20260224_221913.csv` — process-side file with transcripts, response text, and the three creative-direction ratings.
- `creativity_scored_20260316_192451.csv` — product-side creativity scores.
- `human_ratings.csv` — item-level human validation ratings (`condition=process` and `condition=product`).
- `aggregates_master_final.csv` — person-level covariates and external measures used in the mediation analyses.

Join key: `participant_id`, `task_num`

### `sample3/`

- `sample3_analysis.csv` — main processed file with transcripts, responses, creative-direction scores, creativity scores, and exploratory covariates.
- `gptzero_predictions.csv` — bundled GPTZero predictions for the preregistered exploratory analysis.
- `gptzero_summary.json` — summary statistics from the GPTZero analysis.
- `within_group_similarity.csv` — mean pairwise cosine similarity to other products in the same within-task creative-direction group; regenerated by `scripts/homogenization_s3.py`.

Join key: `participant_id`, `task_number`

### `sample4/`

Derived from the public WildChat dataset (https://huggingface.co/datasets/allenai/WildChat, ODC-BY).

- `sample4_process_full_20260225_082835.csv` — process-side WildChat file with hashed conversation and user identifiers, model name, task type, turn count, and creative-direction ratings.
- `sample4_creativity_20260306.csv` — product-side creativity scores.
- `wildchat_hashed_ip_lookup.csv` — lookup table to cluster multiple conversations from the same hashed upstream user.

Join key: `conversation_hash`

## Variable Notes

- `mean_creative_direction` — mean of the Anthropic, Google, and Mistral process-side ratings.
- `avg_creativity` — mean of the OpenAI, Qwen, and xAI product-side creativity ratings.
- `n_user_turns` — number of user messages in the transcript.
- `mails_total` — sample-specific AI literacy composite used in exploratory analyses.
- `self_report_cc` — Sample 3 self-report creative-direction composite.
- `within_group_sim` — mean pairwise cosine similarity within a within-task creative-direction group.
